# Ternary Operator
```
condition ? iftrue : iffalse
```

```dart
int num1 = 10;
int num2 = 15;
int max = (num1>num2) ? num1 :num2;

print(max); // ---> 15
```

```dart
var selection = 3;
var output = (selection == 2) ? 'dart' : 'flutter';

print(output); // ---> flutter
```

```dart
var age = 18;
var check = (age >= 18) ? 'adult' : 'not adult';

print(check); // ---> adult
```
