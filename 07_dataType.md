# List

```dart
List<String> names = ['iman', 'farshad', 'ali', 'reza'];

print('values of names is $names'); // ---> ['iman', 'farshad', 'ali', 'reza']
print('values of names[0] is ${names[0]}'); // index 0 ---> iman


List<int> numbers = [1, 2, 3, 4, 5, 6];
int len = numbers.length; // ---> 6
```
